Cho đến bây giờ, chúng ta đã đề cập đến các compute options như EC2, Lambda, và container services như ECS. Nhưng đợi đã... còn nhiều hơn nữa! Có nhiều purpose-built services bạn có thể sử dụng cho specific use cases có thể giúp bạn đạt được mục tiêu của mình. Hãy xem nhanh một số trong số chúng!

AWS cung cấp một loạt các services được xây dựng có mục đích được thiết kế để streamlines specific use cases—từ application deployment và data processing đến web hosting và hybrid cloud setups. Hãy tham quan nhanh một số solutions mạnh mẽ này.

## Mục tiêu học tập

Sau khi hoàn thành bài học này, bạn sẽ có thể:

- ✅ Mô tả cách Elastic Beanstalk streamlines environment provisioning và management
- ✅ Giải thích cách AWS Batch quản lý large-scale computing tasks và tự động điều chỉnh resources dựa trên demand
- ✅ Xác định cách Amazon Lightsail streamlines web application setup và management mà không cần complex infrastructure
- ✅ Mô tả cách AWS Outposts mở rộng AWS services đến on-premises environments, hỗ trợ hybrid cloud architectures

## Additional Compute Services

AWS cung cấp purpose-built services cho specific needs, chẳng hạn như streamlining web application deployment, managing batch workloads, providing virtual servers, và extending cloud infrastructure đến on-premises data centers.

**AWS Elastic Beanstalk** là một fully managed service streamlines deployment, management, và scaling của web applications. Developers có thể upload code của họ, và Elastic Beanstalk tự động xử lý provisioning của infrastructure, scaling, load balancing, và application health monitoring. Nó hỗ trợ various programming languages và frameworks, chẳng hạn như Java, .NET, Python, Node.js, Docker, và nhiều hơn nữa. Nó cung cấp full control over underlying AWS resources trong khi tự động hóa nhiều operational tasks.

### Cách Elastic Beanstalk hoạt động

Thay vì build out infrastructure cần thiết, như network, EC2 instances, scaling, và elastic load balancers bằng chính bạn, bạn có thể cung cấp application code và desired configurations cho Elastic Beanstalk service. Elastic Beanstalk sau đó lấy thông tin đó và build out environment cho bạn.

**Key Features:**
- ✅ **Automatic Infrastructure Provisioning**: Tự động tạo network, EC2 instances, load balancers
- ✅ **Auto Scaling**: Tự động scale dựa trên demand
- ✅ **Health Monitoring**: Tự động monitor application health
- ✅ **Environment Configuration**: Dễ dàng lưu và deploy lại environment configurations
- ✅ **Full Control**: Vẫn có visibility và control của underlying resources
- ✅ **Multi-language Support**: Java, .NET, Python, Node.js, Docker, và nhiều hơn nữa

**Good for:**
- Deploying và managing web applications
- RESTful APIs
- Mobile backend services
- Microservices architectures
- Automated scaling và simplified infrastructure management

**AWS Batch** là một fully managed service bạn có thể sử dụng để chạy batch computing workloads trên AWS. Nó tự động schedules, manages, và scales compute resources cho batch jobs, tối ưu hóa resource allocation dựa trên job requirements.

### Cách AWS Batch hoạt động

AWS Batch là một compute service được thiết kế cho heavy-duty tasks như processing massive datasets, running simulations, hoặc performing complex calculations. AWS Batch chăm sóc infrastructure management cho bạn. Bạn sẽ không cần lo lắng về provisioning servers, scaling resources, hoặc managing infrastructure. AWS Batch xử lý tất cả điều đó, cho phép bạn focus vào các tasks quan trọng như building application hoặc running analysis của bạn.

**Key Features:**
- ✅ **Automatic Resource Management**: Tự động provision và scale resources
- ✅ **Job Scheduling**: Tự động schedule và distribute tasks
- ✅ **Cost Optimization**: Tối ưu hóa resource allocation
- ✅ **No Infrastructure Management**: Không cần quản lý servers hoặc infrastructure
- ✅ **Automatic Scaling**: Tự động scale dựa trên workload

Service cũng scales tự động, distributing tasks trên một fleet của compute resources như EC2 instances.

**Good for:**
- Processing large-scale, parallel workloads
- Scientific computing
- Financial risk analysis
- Media transcoding
- Big data processing
- Machine learning training
- Genomics research

**Amazon Lightsail** là một cloud service offering virtual private servers (VPSs), storage, databases, và networking tại một predictable monthly price. Nó lý tưởng cho small businesses, basic workloads, và developers tìm kiếm một straightforward AWS experience mà không có complexity của full AWS Management Console.

### Cách Amazon Lightsail hoạt động

Amazon Lightsail simplifies web application hosting bằng cách cung cấp cho bạn một relatively easy, cost-effective solution để chạy specific types của applications và websites. Nó chăm sóc rất nhiều complexity đi kèm với traditional web application hosting, và nó là một great option nếu bạn muốn một cái gì đó quick và easy để quản lý!

**Key Features:**
- ✅ **Virtual Private Servers (VPS)**: Pre-configured VPS instances
- ✅ **Predictable Pricing**: Monthly pricing đơn giản và dự đoán được
- ✅ **Simplified Management**: Dễ sử dụng, không cần complex configurations
- ✅ **All-in-one Solution**: VPS, storage, databases, và networking trong một package
- ✅ **Quick Setup**: Get started nhanh chóng

**Good for:**
- Basic web applications
- Low-traffic websites
- Development và testing environments
- Small business websites
- Blogs
- Learning cloud services

**AWS Outposts** là một fully managed hybrid cloud solution mở rộng AWS infrastructure và services đến on-premises data centers. Nó cung cấp một consistent experience giữa on premises và AWS Cloud, offering compute, storage, và networking components.

### Cách AWS Outposts hoạt động

AWS Outposts được thiết kế cho organizations cần một hybrid-cloud solution. Nếu bạn muốn leverage power của AWS trong khi giữ một số infrastructure của bạn on premises, Outposts là câu trả lời. Outposts mở rộng AWS services đến on-premises data center của bạn, cung cấp cho bạn một consistent experience trên cả hai environments. Bạn có thể chạy AWS services locally, trong khi vẫn hưởng lợi từ cloud computing.

**Key Features:**
- ✅ **Hybrid Cloud**: Kết hợp on-premises và cloud
- ✅ **Consistent Experience**: Same APIs, tools, và services như AWS Cloud
- ✅ **Fully Managed**: AWS quản lý infrastructure
- ✅ **Low Latency**: Chạy applications locally với low latency
- ✅ **Data Residency**: Giữ data on-premises khi cần

**Good for:**
- Low-latency applications
- Data processing trong remote locations
- Migrating và modernizing legacy applications
- Meeting regulatory compliance hoặc data residency requirements
- Hybrid deployments

## So sánh các Additional Compute Services

| Service | Use Case | Key Benefit | Management Level |
|---------|----------|-------------|------------------|
| **Elastic Beanstalk** | Web application deployment | Automated infrastructure provisioning | Managed (với full control) |
| **AWS Batch** | Large-scale batch computing | Automatic resource management | Fully managed |
| **Amazon Lightsail** | Simple web hosting | Easy setup, predictable pricing | Simplified management |
| **AWS Outposts** | Hybrid cloud | Consistent AWS experience on-premises | Fully managed |

## Tóm tắt

Từ Elastic Beanstalk đến Outposts. Mỗi service được thiết kế để xử lý specific tasks, vì vậy hãy nhớ điều đó khi bạn học về các services khác và use cases của chúng.

Trong bài học này, bạn đã học về:

- ✅ **AWS Elastic Beanstalk**: Streamlines web application deployment với automated infrastructure provisioning
- ✅ **AWS Batch**: Fully managed service cho large-scale batch computing workloads
- ✅ **Amazon Lightsail**: Simplified VPS service với predictable pricing
- ✅ **AWS Outposts**: Hybrid cloud solution mở rộng AWS đến on-premises

## Test your skills

### Câu hỏi 1

**Match each AWS service with the correct description:**

**Services:**
1. AWS Batch
2. Amazon Lightsail
3. AWS Elastic Beanstalk
4. AWS Outposts

**Descriptions:**
- A managed service for deploying and scaling web applications
- A fully managed service for batch computing workloads
- A simplified service with virtual private servers (VPSs), storage, and networking
- A hybrid cloud service extending AWS to on premises

**Đáp án:**
- **1 → A fully managed service for batch computing workloads**: AWS Batch chạy large-scale batch computing workloads.
- **2 → A simplified service with virtual private servers (VPSs), storage, and networking**: Amazon Lightsail cung cấp simplified VPS, storage, và networking.
- **3 → A managed service for deploying and scaling web applications**: AWS Elastic Beanstalk là managed service để deploy và scale web applications.
- **4 → A hybrid cloud service extending AWS to on premises**: AWS Outposts mở rộng AWS services đến on-premises environments.

**Giải thích**: AWS Elastic Beanstalk là managed service để deploy và scale web applications. AWS Batch chạy large-scale batch computing workloads. Amazon Lightsail cung cấp simplified VPS, storage, và networking. AWS Outposts mở rộng AWS services đến on-premises environments.

### Câu hỏi 2

**A startup is building a web application and wants a simple way to deploy code without managing infrastructure. They need to have auto scaling and monitoring handled for them.**

**Which AWS service should they use?**

- [x] **AWS Elastic Beanstalk** ← Correct
- [ ] Amazon EC2
- [ ] AWS Batch
- [ ] AWS Outposts

**Giải thích**: AWS Elastic Beanstalk là perfect choice cho startup này vì nó:
- Cho phép deploy code mà không cần quản lý infrastructure
- Tự động xử lý auto scaling
- Tự động xử lý monitoring
- Streamlines deployment process

AWS Outposts được sử dụng để mở rộng AWS đến on-premises environments, không phải để simplify web application deployment.

### Câu hỏi 3

**A financial company must run workloads on premises because of strict compliance requirements, but it wants to use AWS services and tools consistently across environments.**

**Which service meets their needs?**

- [ ] AWS Elastic Beanstalk
- [ ] Amazon Lightsail
- [ ] AWS Batch
- [x] **AWS Outposts** ← Correct

**Giải thích**: AWS Outposts là service phù hợp vì:
- Cho phép chạy AWS services on-premises
- Cung cấp consistent experience giữa on-premises và cloud
- Đáp ứng compliance requirements (data residency)
- Fully managed hybrid cloud solution

Elastic Beanstalk không hỗ trợ on-premises deployments—nó chỉ chạy trên AWS Cloud.
